Ching-Yi Lai 等人的 [UMCL: Unimodal-generated Multimodal Contrastive Learning for Cross-compression-rate Deepfake Detection](http://arxiv.org/abs/2511.18983v1 ) 提出了一种用于跨压缩率（CCR）深度伪造检测的新型单模态生成多模态对比学习框架。**该框架通过将单模态视觉数据转换为三种互补特征（抗压缩的rPPG信号、时间动态特征和预训练视觉语言模型的语义嵌入），并通过亲和力驱动的语义对齐（ASA）策略显式对齐这些特征**。此外，跨质量相似性学习（CQSL）策略进一步增强了特征在不同压缩率下的鲁棒性。实验表明，该方法在多种压缩率和伪造类型下均表现出色，即使单个特征退化也能保持高检测精度。该成果于2024年9月发表于 IJCV。

[Real-Time Mobile Video Analytics for Pre-arrival Emergency Medical Services](http://arxiv.org/abs/2511.14119v1 ) 提出了TeleEMS系统，用于急救医疗服务（EMS）中的实时移动视频分析。**该系统融合音频和视频数据，通过EMS-Stream通信骨干支持多方视频流，并集成了三个实时分析模块**：基于领域专用LLM（EMSLlama）的音频-症状分析、基于rPPG方法的视频-生命体征分析，以及通过多模态多任务模型（PreNet）的联合文本-生命体征分析。实验表明，EMSLlama在症状提取上优于GPT-4o，且多模态融合提升了推断鲁棒性。

Ba-Thinh Nguyen 等人的 [Reperio-rPPG: Relational Temporal Graph Neural Networks for Periodicity Learning in Remote Physiological Measurement](http://arxiv.org/abs/2511.05946v1 ) 提出了一种新型rPPG框架，**通过关系卷积网络和图变换器捕获生理信号的周期性结构**，并设计了针对性的CutMix数据增强提升泛化性。在PURE、UBFC-rPPG和MMPD数据集上的实验表明，该方法在各种运动和光照条件下均具有优异的鲁棒性和性能。

Taixi Chen 等人的 [TYrPPG: Uncomplicated and Enhanced Learning Capability rPPG for Remote Heart Rate Estimation](http://arxiv.org/abs/2511.05833v1 ) 提出了一种基于Mambaout结构的高效rPPG算法。**核心创新包括门控视频理解块（GVB）和综合监督损失函数（CSL）**，GVB结合2D/3D-CNN增强视频分析能力。实验表明，该方法在常用数据集上达到SOTA性能，适用于远程心率估计。该成果于2025年发表于 IEEE WI-IAT 的AI4SG研讨会。

Xulin Ma 等人的 [Non-Contact Health Monitoring During Daily Personal Care Routines](http://arxiv.org/abs/2506.09718v2 ) 发布了首个长期rPPG数据集LADH，包含21名参与者在5种日常护理场景下的同步RGB/红外视频及生理信号。**研究表明，结合RGB和红外输入可提升心率估计精度（MAE=4.99 BPM），且多任务学习能同时优化多项生理指标**。该成果于2025年发表于 IEEE BSN。

Fangling Jiang 等人的 [Learning Knowledge-based Prompts for Robust 3D Mask Presentation Attack Detection](http://arxiv.org/abs/2505.03610v2 ) 提出了一种基于知识图谱提示学习的新型框架，用于3D面具呈现攻击检测。**该方法通过融入知识图谱实体和三联体生成细粒度的显式提示，并引入视觉特定的知识过滤器来优化相关元素**。此外，利用因果图理论消除虚假相关性，增强了模型的泛化能力。实验表明该方法在基准数据集上取得了最优的跨场景检测性能。该成果于2025年10月发表于 TPAMI。

Zahra Maleki 等人的 [SkinMap: Weighted Full-Body Skin Segmentation for Robust Remote Photoplethysmography](http://arxiv.org/abs/2510.05296v1 ) 提出了一种新颖的皮肤分割技术，通过**加权全身皮肤区域检测**提升rPPG信号质量。该方法能有效去除干扰区域（如嘴部、眼睛和头发），并在运动、说话等挑战性条件下保持心率估计准确性。作者还发布了新数据集SYNC-rPPG以更好地模拟真实场景，验证了模型对多种肤色的高鲁棒性。

Tianwen Zhou 等人的 [Editing Physiological Signals in Videos Using Latent Representations](http://arxiv.org/abs/2509.25348v2 ) 提出了一种基于隐式表示的生理信号编辑框架。**通过3D VAE编码视频并结合目标心率提示，利用自适应层归一化和FiLM调制实现视频中心率信号的精确编辑**。该方法在保持视觉质量（PSNR 38.96 dB，SSIM 0.98）的同时，实现了10 bpm MAE的心率调制误差，适用于生物信号匿名化或合成具有特定生命体征的视频。

Shuyang Chu 等人的 [To Remember, To Adapt, To Preempt: A Stable Continual Test-Time Adaptation Framework for Remote Physiological Measurement in Dynamic Domain Shifts](http://arxiv.org/abs/2510.01282v1 ) 提出了名为PhysRAP的持续测试时适应框架。**通过冻结生理相关参数防止灾难性遗忘，并采用预emptive梯度修改策略避免过适应**。该方法在动态域偏移下表现出色，代码已开源。

Bo Zhao 等人的 [PHASE-Net: Physics-Grounded Harmonic Attention System for Efficient Remote Photoplethysmography Measurement](http://arxiv.org/abs/2509.24850v2 ) 从流体力学纳维-斯托克斯方程出发，推导出脉搏信号遵循二阶动态系统的理论模型。基于此设计了**零计算量轴向交换模块、自适应空间滤波器和门控TCN**，实现了高效且理论解释性强的rPPG测量，在轻量化模型中达到SOTA性能。

Constantino Álvarez Casado 等人的 [Design, Implementation and Evaluation of a Real-Time Remote Photoplethysmography (rPPG) Acquisition System for Non-Invasive Vital Sign Monitoring](http://arxiv.org/abs/2508.18787v1 ) 提出了一种优化用于低功耗设备的实时远程光电容积描记（rPPG）系统。该系统基于Face2PPG流水线，通过多线程架构管理视频捕获、实时处理、网络通信和图形用户界面（GUI）更新，**能够在30帧/秒的速率下连续可靠运行**，并支持HTTP视频流和RESTful API接口。研究采用混合编程模型（FRP和Actor模型）以降低计算开销，为现代医疗和人机交互应用提供了实时生物信号监测的实用解决方案。

Konstantin Egorov 等人的 [Gaze into the Heart: A Multi-View Video Dataset for rPPG and Health Biomarkers Estimation](http://arxiv.org/abs/2508.17924v1 ) 提出了一个面向rPPG和健康指标估计的大规模多视角视频数据集。该数据集包含**600名受试者在静息和运动后状态下的3600段同步视频**，搭配100Hz PPG信号及心电图、血压、血氧等多模态生理指标。研究还训练了一个高效rPPG模型，在跨数据集测试中验证了性能。该成果于2025年9月发表于 ACMMM 2025（Datasets track）。

Jiankai Tang 等人的 [Contact Sensors to Remote Cameras: Quantifying Cardiorespiratory Coupling in High-Altitude Exercise Recovery](http://arxiv.org/abs/2508.00773v1 ) 研究了高海拔环境下运动恢复期的心肺耦合（CRC）特征。实验发现恢复期CRC同步事件更频繁但稳定性降低（p < 0.05），并验证了**基于rPPG的非接触式CRC测量与血氧仪结果高度相关（Pearson r=0.96）**。该成果于2025年9月发表于 UbiComp 25。

Joaquim Comas 等人的 [BeatFormer: Efficient motion-robust remote heart rate estimation through unsupervised spectral zoomed attention filters](http://arxiv.org/abs/2507.14885v1 ) 提出了一种轻量级频谱注意力模型BeatFormer。该模型结合**缩放正交复数注意力机制和频域能量测量**，并通过无监督频谱对比学习（SCL）实现免标签训练，在PURE、UBFC-rPPG等数据集上展现了优异运动鲁棒性。

Kang Cen 等人的 [Robust and Generalizable Heart Rate Estimation via Deep Learning for Remote Photoplethysmography in Complex Scenarios](http://arxiv.org/abs/2507.07795v1 ) 设计了一个端到端rPPG提取网络。通过**差分帧融合模块和时序移位模块（TSM）**增强脉搏波特征表示，配合动态混合损失函数，在MMPD跨数据集测试中达到7.58 MAE，优于现有最优模型。

Zhipeng Li 等人的 [Exploring Remote Physiological Signal Measurement under Dynamic Lighting Conditions at Night: Dataset, Experiment, and Analysis](http://arxiv.org/abs/2507.04306v1 ) 发布了首个夜间动态光照条件下的rPPG数据集DLCN，包含**98名受试者在4类典型夜间场景下13小时视频及生理信号**。基于Happy-rPPG工具包的实验揭示了现有方法在复杂光照环境中的性能瓶颈。

Jiho Choi 等人的 [Periodic-MAE: Periodic Video Masked Autoencoder for rPPG Estimation](http://arxiv.org/abs/2506.21855v1 ) 提出了一种通过自监督学习从无标记面部视频中提取周期性信号（如皮肤色调的细微变化）的通用表示方法。该框架采用视频掩码自编码器（MAE）学习面部区域的高维时空表示，**创新性地引入帧掩码采样和生理频带限制约束**，以增强对远程光电容积描记（rPPG）信号的建模能力。实验表明，该方法在PURE、UBFC-rPPG等数据集上显著提升了跨数据集性能。

Jiyao Wang 等人的 [Align the GAP: Prior-based Unified Multi-Task Remote Physiological Measurement Framework For Domain Generalization and Personalization](http://arxiv.org/abs/2506.16160v1 ) 提出了一种统一框架GAP，**同时解决多源跨域泛化（MSSDG）和测试时个性化适应（TTPA）问题**。通过解耦面部视频中的不变语义、个体偏差和噪声，结合生理先验知识优化模型，该框架在多项生理指标（如rPPG）任务中表现优异，并在新构建的真实驾驶数据集上验证了有效性。

Jiyao Wang 等人的 [Efficient Mixture-of-Expert for Video-based Driver State and Physiological Multi-task Estimation in Conditional Autonomous Driving](http://arxiv.org/abs/2410.21086v2 ) 开发了多任务驾驶员监控系统VDMoE，**利用RGB视频和rPPG信号联合检测认知负荷与疲劳状态**。通过混合专家（MoE）框架整合多模态输入，并引入先验正则化方法加速收敛。实验基于新数据集MCDD和公开数据集，验证了模型在SAE L2/3自动驾驶场景中的高效性和准确性。

Jitesh Joshi 等人的 [Efficient and Robust Multidimensional Attention in Remote Physiological Sensing through Target Signal Constrained Factorization](http://arxiv.org/abs/2505.07013v1 ) 提出了目标信号约束分解模块（TSFM），**将生理信号特征显式融入注意力机制**，并设计了双分支3D-CNN架构MMRPhys，支持从RGB和热成像视频中同步估计rPPG与呼吸信号。跨数据集测试表明，该方法在域偏移下具有更强的鲁棒性，且满足实时性要求。

Yiping Xie 等人的 [PhysLLM: Harnessing Large Language Models for Cross-Modal Remote Physiological Sensing](http://arxiv.org/abs/2505.03621v1 ) 首次将大语言模型（LLM）引入rPPG领域，提出协作优化框架PhysLLM。**通过文本原型引导（TPG）实现跨模态对齐**，并开发双域稳态算法（DDS）抑制信号不稳定。该模型在光照变化和运动场景中展现出卓越的泛化能力，在四个基准数据集上达到SOTA性能。

Kegang Wang 等人的 [Memory-efficient Low-latency Remote Photoplethysmography through Temporal-Spatial State Space Duality](http://arxiv.org/abs/2504.01774v2 ) 提出了一种基于**时空状态空间对偶性**的高效远程光电容积描记（rPPG）算法ME-rPPG。该方法解决了模型可扩展性、跨数据集泛化能力和实时性之间的权衡问题，通过可迁移的状态空间高效捕捉面部视频帧中的细微周期性变化，同时保持极低计算开销。实验表明，ME-rPPG在多个数据集上实现了21.3%-60.2%的性能提升，**仅需3.6 MB内存和9.46 ms延迟**即可完成实时推断。

Rufei Ma 等人的 [RF-BayesPhysNet: A Bayesian rPPG Uncertainty Estimation Method for Complex Scenarios](http://arxiv.org/abs/2504.03915v1 ) 首次将贝叶斯神经网络引入rPPG领域，提出**融合鲁棒性的贝叶斯生理网络**（RF-BayesPhysNet）。该模型通过变分推理同时建模偶然性和认知不确定性，在UBFC-RPPG数据集上取得2.56 MAE的优异表现。论文创新性地提出使用Spearman相关系数、预测区间覆盖率和置信区间宽度等指标评估不确定性估计效果，显著提升了复杂场景下的测量可靠性。该成果于2025年4月发表于机器学习领域会议（根据arxiv:comment提示为11页论文）。

Banafsheh Adami 等人的 [rPPG-SysDiaGAN: Systolic-Diastolic Feature Localization in rPPG Using Generative Adversarial Network with Multi-Domain Discriminator](http://arxiv.org/abs/2504.01220v1 ) 开发了一种**多判别器生成对抗网络**，专注于从面部视频中提取包含收缩期-舒张期特征的完整rPPG信号。通过结合时域、频域和信号二阶导数的三重判别器，以及方差损失、动态时间规整损失等四种损失函数，该方法有效解决了现有模型对PPG信号完整波形重建不足的问题。

Gyutae Hwang 等人的 [Phase-shifted remote photoplethysmography for estimating heart rate and blood pressure from facial video](http://arxiv.org/abs/2401.04560v4 ) 提出**两阶段深度学习框架**，通过双路rPPG网络（DRP-Net）生成肢端与面部的相位差rPPG信号，再结合有界血压网络（BBP-Net）分析相位差异。该方法在MMSE-HR数据集上实现心率估计MAE 1.78 BPM（较现有方法提升34.31%），血压估计误差SBP 10.19 mmHg/DBP 7.09 mmHg。该成果于2024年1月发表于计算机视觉会议（根据arxiv:comment提示为13页论文）。

Hang Shao 等人的 [Remote Photoplethysmography in Real-World and Extreme Lighting Scenarios](http://arxiv.org/abs/2503.11465v1 ) 设计了一种**端到端视频Transformer模型**，通过全局干扰共享、主体背景参考等创新机制消除极端光照下的时变干扰。该方法首次实现了基于自然面部视频的轻量化户外rPPG监测，其时空滤波和生物先验约束策略显著提升了真实场景下的鲁棒性。

Theodore Curran 等人的 [Estimating Blood Pressure with a Camera: An Exploratory Study of Ambulatory Patients with Cardiovascular Disease](http://arxiv.org/abs/2503.00890v1 ) 提出了一种基于远程光电容积图（rPPG）的非接触式血压监测方法。**该研究首次在心血管疾病高风险人群（包括房颤患者）中验证了rPPG技术的可行性**，通过深度学习模型分析面部视频信号，结果显示房颤患者的血压预测准确性与窦性心律患者相当，为高血压监测提供了新思路。

Bochao Zou 等人的 [RhythmMamba: Fast, Lightweight, and Accurate Remote Physiological Measurement](http://arxiv.org/abs/2404.06483v2 ) 提出了一种基于状态空间模型（Mamba）的rPPG信号处理方法。**通过将rPPG视为时间序列任务并设计多时序约束和频域前馈机制**，该方法在保持线性计算复杂度的同时，实现了319%的吞吐量提升和23%的GPU内存占用降低，显著优于现有技术。

Bochao Zou 等人的 [RhythmFormer: Extracting Patterned rPPG Signals based on Periodic Sparse Attention](http://arxiv.org/abs/2402.12788v3 ) 提出了一种周期性稀疏注意力机制。**通过预注意力阶段学习rPPG信号的周期模式，过滤无关计算**，并结合融合模块增强特征鲁棒性，在跨数据集评估中达到最优性能，解决了传统Transformer在长序列处理中的计算效率问题。

Zheng Wu 等人的 [CardiacMamba: A Multimodal RGB-RF Fusion Framework with State Space Models for Remote Physiological Measurement](http://arxiv.org/abs/2502.13624v1 ) 开发了一种融合RGB与射频（RF）信号的多模态心率估计框架。**通过时序差分Mamba模块和双向状态空间模型**，该方法在EquiPleth数据集上显著降低了肤色偏差，并在模态缺失场景中保持鲁棒性，推动了rPPG技术的公平性应用。

Zijie Yue 等人的 [Bootstrapping Vision-language Models for Self-supervised Remote Physiological Measurement](http://arxiv.org/abs/2407.08507v2 ) 提出了一种基于视觉语言模型（VLM）的自监督rPPG学习框架。**通过频率导向的视觉-文本对生成和对比学习机制**，该方法在无监督条件下实现了与监督方法接近的性能，为突破rPPG数据标注瓶颈提供了新方案。该成果于2024年9月发表于 International Journal of Computer Vision。

Bingjie Wu 等人的 [Semi-rPPG: Semi-Supervised Remote Physiological Measurement with Curriculum Pseudo-Labeling](http://arxiv.org/abs/2502.03855v1 ) 提出了一种结合课程伪标记和一致性正则化的半监督学习方法，用于从未标记的面部视频中提取生理特征。**创新点包括基于信噪比（SNR）的自适应课程伪标记策略，以及针对准周期信号的增强一致性正则化**。该方法在四个公共数据集上通过跨数据集评估展示了优越性能。该成果于2024年9月发表于 IEEE Transactions on Instrumentation and Measurement (TIM)。

Alexey Protopopov 等人的 [A Robust Remote Photoplethysmography Method](http://arxiv.org/abs/2502.02229v1 ) 提出了一种对运动和环境光照干扰更具鲁棒性的rPPG方法。**通过结合多种数学变换计算心率，该方法在未修改的红外相机上表现最佳，平均绝对误差低至1.95 BPM**。实验基于19名志愿者的26段视频，验证了其在真实场景下的准确性优势。

Yuting Zhang 等人的 [Advancing Generalizable Remote Physiological Measurement through the Integration of Explicit and Implicit Prior Knowledge](http://arxiv.org/abs/2403.06947v2 ) 提出了一种融合显式与隐式先验知识的框架，以提升rPPG的跨域泛化能力。**显式部分系统分析了不同域（如摄像头、光照、肤色）的噪声源，隐式部分通过双分支网络解耦生理特征与噪声**。实验表明该方法在RGB到近红外（NIR）数据集的迁移中也具有优越表现。

### 主要研究方向

1. **多模态融合与跨域泛化**
  - 研究方向概述：通过结合多种模态（如RGB、红外、rPPG信号等）和跨域学习方法，提升rPPG在不同环境和设备下的鲁棒性。
  - 代表性研究
    - *[UMCL: Unimodal-generated Multimodal Contrastive Learning for Cross-compression-rate Deepfake Detection](http://arxiv.org/abs/2511.18983v1 )* (2024年9月发表于IJCV)
    - *[Align the GAP: Prior-based Unified Multi-Task Remote Physiological Measurement Framework For Domain Generalization and Personalization](http://arxiv.org/abs/2506.16160v1 )* (2025年发表于IEEE BSN)

2. **高效轻量化模型设计**
  - 研究方向概述：优化rPPG模型的计算效率和实时性，适用于移动设备和低功耗场景。
  - 代表性研究
    - *[TYrPPG: Uncomplicated and Enhanced Learning Capability rPPG for Remote Heart Rate Estimation](http://arxiv.org/abs/2511.05833v1 )* (2025年发表于IEEE WI-IAT)

3. **动态环境与噪声鲁棒性**

### 研究趋势分析

过去几年，rPPG研究呈现出以下发展趋势：
1. **从单模态到多模态融合**：早期研究主要依赖RGB视频，近年来逐渐引入红外、射频（RF）等多模态数据，提升信号鲁棒性。
2. **从监督学习到自监督/无监督学习**：为解决标注数据稀缺问题，自监督和无监督方法（如MAE、对比学习）成为研究热点。
3. **从静态环境到动态场景**：研究重点从实验室环境转向真实复杂场景（如运动、光照变化），注重动态噪声抑制。
4. **从单一心率估计到多生理指标**：rPPG应用从心率扩展到血压、呼吸、血氧等多指标联合估计。
5. **轻量化与实时性**：模型设计趋向高效轻量化，以满足移动设备和实时监测需求。
6. **跨域泛化与临床验证**：研究更加注重跨数据集和跨场景性能，并在医疗场景中验证技术可行性。
