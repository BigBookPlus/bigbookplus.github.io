Pengyu Chen 等人的 [CoDrone: Autonomous Drone Navigation Assisted by Edge and Cloud Foundation Models](http://arxiv.org/abs/2512.19083v2 ) 提出了首个云-边-端协同计算框架CoDrone，**将基础模型集成到无人机自主巡航场景中**。该框架使用灰度图像进行导航，在需要增强环境感知时利用边缘辅助的Depth Anything V2模型进行深度估计，并提出**基于一维占据网格的新型导航方法**。实验表明，CoDrone在动态环境中表现优异，平均飞行距离增加40%，导航质量提高5%。该成果发表于IEEE Internet of Things Journal。

Qianxue Zhang 等人的 [A Multicore and Edge TPU-Accelerated Multimodal TinyML System for Livestock Behavior Recognition](http://arxiv.org/abs/2504.11467v3 ) 开发了一种基于TinyML技术的牲畜行为识别系统。**该系统融合加速度计数据和视觉输入**，构建了用于图像分类、目标检测和行为识别的多模态网络，在商用微控制器上实现了实时推理，**模型大小减少270倍，响应延迟低于80ms**。结合无线通信技术，该系统为偏远地区的牲畜监测提供了灵活、可扩展的解决方案。该成果发表于IEEE Internet of Things Journal。

Zhenyuan Chen 等人的 [RSCC: A Large-Scale Remote Sensing Change Caption Dataset for Disaster Events](http://arxiv.org/abs/2509.01907v5 ) 提出了一个大规模遥感变化描述数据集RSCC，包含62,351对灾害前后图像及详细文本描述。**该数据集填补了现有资源在时间序列图像和文本注释方面的空白**，为灾害感知的双时相理解提供了训练和评估基准。该数据集被NeurIPS 2025数据集与基准赛道接收。

Brandon Marks 等人的 [Do Generalized-Gamma Scale Mixtures of Normals Fit Large Image Datasets?](http://arxiv.org/abs/2512.17038v2 ) 研究了广义伽马尺度混合正态分布对大型图像数据集的拟合效果。**该研究首次证明该先验模型对多个大型成像数据集具有现实性**，包括遥感、医学成像和图像分类应用。研究表明，该先验家族对数据集的拟合效果显著优于包括高斯、拉普拉斯和Student's t在内的标准先验，**为逆成像问题的贝叶斯推断提供了更优选择**。

Jinghao Shi 等人的 [BiCoR-Seg: Bidirectional Co-Refinement Framework for High-Resolution Remote Sensing Image Segmentation](http://arxiv.org/abs/2512.20255v1 ) 提出了一个双向协同优化框架BiCoR-Seg，用于解决高分辨率遥感图像语义分割中类间相似性高、类内差异大的挑战。该框架通过**热图驱动的双向信息协同模块（HBIS）**在特征图和类别嵌入之间建立双向信息流，并结合**分层监督策略**增强浅层特征的判别能力。此外，还提出**跨层类别嵌入Fisher判别损失**来提高嵌入表示的可区分性。实验表明，BiCoR-Seg在LoveDA等数据集上实现了优异的性能，同时具有更强的可解释性。

Binfeng Wang 等人的 [Degradation-Aware Metric Prompting for Hyperspectral Image Restoration](http://arxiv.org/abs/2512.20251v1 ) 提出了DAMP框架，用于统一的高光谱图像（HSI）恢复。该框架通过**空间-光谱退化指标**生成连续量化的退化提示（DP），替代传统依赖显式退化先验的方法。**空间-光谱自适应模块（SSAM）**动态调节特征提取，结合混合专家架构实现自适应恢复。DAMP在自然和遥感HSI数据集上表现出色，并展示了强大的泛化能力。

Liang Yao 等人的 [RemoteReasoner: Towards Unifying Geospatial Reasoning Workflow](http://arxiv.org/abs/2507.19280v3 ) 提出了一种统一的地理空间推理工作流RemoteReasoner。该框架通过**多模态大语言模型（MLLM）**解析用户指令和定位目标，结合**任务转换策略**支持对象级、区域级和像素级的多粒度任务。通过强化学习训练，模型能够自主构建推理路径，无需任务特定解码器或微调即可适应多样化任务，在多项任务中达到SOTA性能。

Li Pang 等人的 [SPECIAL: Zero-shot Hyperspectral Image Classification With CLIP](http://arxiv.org/abs/2501.16222v3 ) 提出了一种基于CLIP的零样本高光谱图像分类框架。该框架通过**CLIP生成伪标签**和**噪声标签学习**两阶段实现无需人工标注的分类。通过多尺度预测融合和标签细化技术，SPECIAL在三个基准数据集上超越了现有零样本方法，展现了实际应用潜力。

Zepeng Xin 等人的 [SegEarth-R2: Towards Comprehensive Language-guided Segmentation for Remote Sensing Images](http://arxiv.org/abs/2512.20013v1 ) 提出了LaSeRS数据集和SegEarth-R2模型，专注于复杂语言引导的遥感图像分割。模型通过**空间注意力监督机制**改进小目标定位，并采用**灵活的分割查询机制**处理单目标和多目标场景。实验表明，SegEarth-R2在多层次粒度、多目标指令和隐含意图解析等维度上表现优异。

Georgios Voulgaris 等人的 [FusionNet: Physics-Aware Representation Learning for Multi-Spectral and Thermal Data via Trainable Signal-Processing Priors](http://arxiv.org/abs/2512.19504v1 ) 提出了一种**多光谱与热红外数据融合的物理感知表征学习框架**。通过嵌入可训练的信号处理先验知识，FusionNet结合了短波红外（SWIR）和热红外（TIR）数据，**在跨光谱条件下表现出强大鲁棒性**。该成果于2024年9月发表于 IEEE Journal of Selected Topics in Applied Earth Observations and Remote Sensing (JSTARS)。

Zhenyang Huang 等人的 [ReasonCD: A Multimodal Reasoning Large Model for Implicit Change-of-Interest Semantic Mining](http://arxiv.org/abs/2512.19354v1 ) 设计了一个**多模态推理大模型**，能够从隐含的文本描述中挖掘用户对遥感图像变化检测的意图。**模型利用预训练大语言模型的推理能力**，在BCDD数据集上取得了92.1%的F1分数，同时能够解释推理过程以辅助决策。

Azadeh Alavi 等人的 [Practical Quantum-Classical Feature Fusion for complex data Classification](http://arxiv.org/abs/2512.19180v1 ) 研究了**量子-经典混合学习的跨注意力中融合架构**，通过注意力机制将量子特征与经典特征融合，**在复杂数据集（如Forest CoverType）上表现优于传统混合方法**，展示了量子信息在多模态融合中的价值。

Jin Ye 等人的 [Deep Equilibrium Convolutional Sparse Coding for Hyperspectral Image Denoising](http://arxiv.org/abs/2508.15553v2 ) 提出了一种**基于深度均衡（DEQ）的卷积稀疏编码框架**，统一了局部空间-光谱相关性、非局部自相似性和全局空间一致性，**在高光谱图像去噪任务中优于现有方法**。

Pengxiang Ouyang 等人的 [PMPGuard: Catching Pseudo-Matched Pairs in Remote Sensing Image-Text Retrieval](http://arxiv.org/abs/2512.18660v1 ) 提出了一种针对遥感图像-文本检索中**伪匹配对（PMPs）问题**的新型框架。该研究通过**跨模态门控注意力机制**和**正负感知注意力机制**，动态过滤语义不匹配或弱对齐的噪声数据，显著提升了跨模态对齐的鲁棒性。实验在RSICD、RSITMD和RS5M三个基准数据集上验证了其**最先进的性能**。

Oraib Almegdadi 等人的 [Self-organizing maps for water quality assessment in reservoirs and lakes: A systematic literature review](http://arxiv.org/abs/2512.18466v1 ) 系统综述了**自组织映射（SOM）**在湖泊与水库水质评估中的应用。研究总结了参数选择、时空采样策略和聚类方法，**强调SOM在无标签数据下处理高维非线性关系的能力**，尤其在生态评估、富营养化分类和藻华监测中的有效性。该成果已被期刊 *Ecological Informatics* 录用（DOI:10.1016/j.ecoinf.2025.103542）。

Danilo Costarelli 等人的 [Implementation of neural network operators with applications to remote sensing data](http://arxiv.org/abs/2412.00375v2 ) 提出了基于**双曲正切激活的神经网络算子算法**，用于遥感图像的建模与超分辨率重建。实验表明，该算法在**结构相似性指数（SSIM）**上优于双线性/双三次插值，尤其适用于RETINA数据集中的遥感图像增强。

Yunkai Dang 等人的 [A Benchmark for Ultra-High-Resolution Remote Sensing MLLMs](http://arxiv.org/abs/2512.17319v1 ) 构建了首个**超高分遥感多模态大模型（MLLM）基准RSHR-Bench**，包含5,329张边长≥4,000像素的图像，涵盖9类感知任务和4类推理任务。研究揭示了现有MLLMs在超高分辨率场景下的性能缺陷，并提供了对抗性过滤的3,864个VQA和3,913个图像描述任务数据。代码已开源。

Jianan Pan 等人的 [Holistic Information Theory of Spatial Remote Sensing Imaging](http://arxiv.org/abs/2512.17163v1 ) 提出了一种空间遥感成像的整体信息理论。该研究**将光电成像硬件前端和计算重建后端整合到一个统一框架中**，建立了以获取有效信息为目标的完整空间成像链信息传输模型。论文创新性地提出了**可量化的MTF-SNR乘积准则**，证明系统信息传输能力由MTF和SNR的乘积决定，且这些参数可以相互补偿以实现等效信息传输。通过高分辨率地球观测系统的案例验证表明，在相同的重建均方误差条件下，增加时间延迟积分级数可减小光学孔径尺寸并显著降低主镜质量。该理论已成功应用于吉林一号卫星星座的设计，为低成本高分辨率遥感系统提供了新范式。

Pallavi Jain 等人的 [TimeSenCLIP: A Time Series Vision-Language Model for Remote Sensing Using Single-Pixel](http://arxiv.org/abs/2508.11919v2 ) 提出了一种轻量级遥感时间序列视觉语言模型。该模型采用跨视角时间对比框架，**将多光谱Sentinel-2时间序列与地理标记的地面图像对齐，无需文本标注**。与现有VLM不同，TimeSenCLIP**强调时间与光谱信号而非空间上下文**，探索单像素时间序列是否包含足够信息以解决多种任务。该研究为中等分辨率遥感影像的语义理解提供了新思路。

Keiller Nogueira 等人的 [Core-Set Selection for Data-efficient Land Cover Segmentation](http://arxiv.org/abs/2505.01225v3 ) 研究了六种基于图像、标签或其组合的核心集选择方法，用于识别能够保持或超越完整数据集性能的高质量数据子集。实验表明，在DFC2022数据集上，仅使用25%训练数据选出的核心集能使SegFormer性能略高于完整数据集训练结果。该研究**揭示了数据质量对遥感语义分割的重要性**，为数据高效学习建立了基准。

Yunkai Yang 等人的 [Task-Oriented Data Synthesis and Control-Rectify Sampling for Remote Sensing Semantic Segmentation](http://arxiv.org/abs/2512.16740v1 ) 提出任务导向的数据合成框架TODSynth。其核心包括：**统一三重注意力的多模态扩散变换器(MM-DiT)**，以及由任务反馈指导的即插即用采样策略。通过控制-修正流匹配(CRFM)方法动态调整采样方向，**有效缓解生成图像的不稳定性**，缩小合成数据与下游分割任务的差距。实验证明该方法在少样本和复杂场景中显著优于现有可控生成方法。

### 研究趋势分析

过去几年的研究趋势显示，遥感领域的研究重点逐渐从传统的图像处理技术转向深度学习和多模态融合方法。具体表现为：
1. **深度学习主导**：越来越多的研究采用深度学习框架（如双向协同优化、注意力机制、MLLM）解决遥感图像的分割、分类和恢复问题。
2. **多模态融合**：结合视觉、文本和物理先验的多模态方法成为研究热点，尤其在变化检测和灾害评估中表现突出。
3. **轻量化与实时性**：针对农业和生态监测，轻量级模型和半监督方法受到重视，以适应边缘计算和实时应用需求。
4. **数据集与基准推动**：大规模数据集（如RSCC）和基准（如RSHR-Bench）的构建为模型训练和评估提供了重要支持。
5. **跨学科应用**：遥感技术与量子计算、强化学习等新兴领域的结合展现出潜力，如量子-经典混合学习在遥感中的应用。
